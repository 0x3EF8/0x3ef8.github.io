# Linux Desktop Portfolio

A simple Linux-style desktop portfolio built with Next.js.

## What It Includes

- Linux-inspired desktop UI
- Draggable and resizable app surfaces
- File manager-style folders
- Terminal with command support
- Vim-like editor window
- File viewer for supported file types

## Tech Stack

- Next.js 16
- React 19
- TypeScript

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open in browser:

```text
http://localhost:3000
```

## Useful Scripts

```bash
npm run dev
npm run lint
npm test
npm run build
npm run start
```

## Deployment Check

Run these before deployment:

